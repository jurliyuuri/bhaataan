boríṣán báni

現代バート語の古めの表記。borúḷ という用言の意味は今から考える。

gháyo

大文字で転写する。歴史的経緯の情報を含み、それだけでは現代音を必ずしも読み取れないため、あくまでそこに「美しさ」を流し込んで完成形とするための「鋳型」とみなされている。

特殊字母

Q: 直後の子音を伸ばす。

Ъ: 母音のないことを示す。

母音字

I → i U → u Í → í Ú → ú
É → e Ó → o
È → ai Ò → au
(A) → a Á → á

「A は普通は綴らないが、Q の直前では綴る」…としておくのが無難かなぁ？　改正後（つまり、現行仕様）での ~ 様記号との一貫性のために。

子音字

赤背景は外来語

P → p, ph T → t, ṭ Z → z K → k, kh ʔ → Ø
F → p, ph C → c, ṣ X → x, ṣ H → h
B → bh D → d, dh, ḍ Ж → j, gh, y G → gh, g, j, ◌́
V → bh, w S → s, r J → j, gh, y
M → m, b N → n, ṇ Ɲ → ṇ, n Ŋ → ṇ, n
W → w L → l, ḷ R → r

R は基本的に語頭か Q の後にしか来ないはず（基本的に語中の r は短母音 + rr → 長母音 + r であるか S が転じたものであるため）。ただし -lor- 語幹については -LÚR- である。

hámí

「鋳型」gháyo に、「美しさ」を流し込んで完成形とすることで、現代音を表すのに必要な情報が揃う。

転写では、一貫して下につく記号として表せるように、IPA や extIPA などの記号を援用して以下のように転写することとする。

lowered ◌̞ (U+031E)
advanced ◌̟ (U+031F)
retracted ◌̠ (U+0320)
strong articulation ◌͈ (U+0348)
weak articulation ◌͉ (U+0349)
短い鼻音への縮約 ◌Ъ◌
長い鼻音への縮約 ◌Ъ◌

具体例

boríṣán báni は非常に歴史に引きずられているため、これを使うと同根語や関連語の間の関係が見出しやすい。

後置格/無格 ṣaghi「上」と主格 ṣá「上が」

C̠GI → ṣaghi
  1. C には retracted の記号がついているので ṣ
  2. 母音字が付加されていないので a
  3. G は gh
  4. I は i
ゆえに ṣaghi
C̠G͉Ъ → ṣá
  1. C には retracted の記号がついているので ṣ
  2. 母音字が付加されていないので a
  3. G は weak articulation がついているので ◌́（これは、音節末であり直前が短母音であることが原因）
  4. Ъ はゼロ
ゆえに ṣá
このように、boríṣán báni で書くと、主格 ṣá と後置格/無格 ṣaghi が同じ語であることが分かりやすい。

後置格/無格 bhoghi (正書法上は bhogi)「色」と主格 bhog「色が」

BÓGI → bhoghi
  1. B は bh
  2. Ó は o
  3. G は gh
  4. I は i
ゆえに bhoghi (ただし現代正書法上は bhogi と綴る)
BÓG͈Ъ → bhog
  1. B は bh
  2. Ó は o
  3. G は strong articulation がついているので g（これは、音節末であり直前が長母音であることが原因）
  4. Ъ はゼロ
ゆえに bhog
このように、boríṣán báni で書くと、主格 bhog と後置格/無格 bhoghi (正書法上は bhogi) が同じ語であることが分かりやすい。

lom「三」と ṣánom「八」

LÚ̞MЪ → lom
  1. L は語頭なので l
  2. Ú には lowered の記号がついているので o（これは l の後だからである）
  3. M は m
  4. Ъ はゼロ
ゆえに lom
C̠ÁNЪLÚ̞MЪ → ṣánom
  1. C には retracted の記号がついているので ṣ
  2. Á は Á
  3. NЪL には一重線が引いてあるので短い鼻音。L の調音位置によりこれは n
  4. Ú には lowered の記号がついているので o（これは l の後だからである）
  5. M は m
  6. Ъ はゼロ
ゆえに ṣánom
このように、boríṣán báni で書くと、数詞 ṣán「5」と数詞 lom「3」を合わせたものが数詞 ṣánom「8」であることが分かりやすい。

com「どの？」と k「もの」と ká「もの（複数）」と coṇá「どれ？」

CÓMЪ → com
  1. C は c
  2. Ó は o
  3. M は m
  4. Ъ はゼロ
ゆえに com
KЪ → k
  1. K は k
  2. Ъ はゼロ
ゆえに k
KG͉Ъ → ká
  1. K は k
  2. 母音字が付加されていないので a
  3. G は weak articulation がついているので ◌́（これは、音節末であり直前が短母音であることが原因）
  4. Ъ はゼロ
ゆえに ká
CÓMЪKGЪ → coṇá
  1. C は c
  2. Ó は o
  3. MЪK には一重線が引いてあるので短い鼻音。K の調音位置によりこれは ṇ
  4. 母音字が付加されていないので a
  5. G は weak articulation がついているので ◌́（これは、音節末であり直前が短母音であることが原因）
  6. Ъ はゼロ
ゆえに coṇá
このように、boríṣán báni で書くと、疑問名詞 coṇá「どれ？」は形式名詞 k「もの」に複数語尾 -á をつけ疑問連体詞 com「どの？」で修飾したという構成であることが見てとれるとともに、バート語の複数接辞 -á とパイグ語の複数接辞 ge【等】との間の関連も窺い知ることができる。

変幻自在な唇音

BÉ̞TÚLЪ → bhátúḷ
  1. B は bh
  2. É には lowered の記号がついているので á（これは唇音の後だからである）
  3. T は t
  4. Ú は ú
  5. L は前舌母音にも l にも語頭にも隣接していないので ḷ
  6. Ъ はゼロ
ゆえに bhátúḷ
CÓB̠I → codhi
  1. C は c
  2. Ó は o
  3. B には retracted の記号がついているので dh（これは i の前だからである）
  4. I は i
ゆえに codhi
このように、boríṣán báni で書くと、
  • バート語 bhátúḷ は現代リパライン語 phertars と関連する
  • バート語 codhi はパイグ語 cuop2 と関連する
といったことが分かりやすい。

リパライン語からの早期借用

VÉ̞NЪXJÚLЪ → bháṇajúḷ
  1. V は bh
  2. É には lowered の記号がついているので á（これは唇音の後だからである）
  3. NЪX には一重線が引いてあるので短い鼻音。X の調音位置によりこれは ṇ
  4. 母音字が付加されていないので a
  5. J は j
  6. Ú は ú
  7. L は前舌母音にも l にも語頭にも隣接していないので ḷ
  8. Ъ はゼロ
ゆえに bháṇajúḷ
VAVЪC̠ → bhauṣ
  1. V は bh
  2. AVЪ は au
  3. C には retracted の記号がついているので ṣ（これは語幹末だからである）
  4. Ъ はゼロ
ゆえに bhauṣ
このように、boríṣán báni で書くと、
  • バート語 bháṇajúḷ は現代リパライン語 fenxe に相当する語の早期借用
  • バート語 bhauṣ は現代リパライン語 fafs に相当する語の早期借用
であることが分かりやすい。

nautu

Ɲ̟ÒTU → nautu
  1. Ɲ には advanced の記号がついているので n（これは語頭だからである）
  2. Ò は au
  3. T は t
  4. U は u
ゆえに nautu
LÍŊƝÒT̠U → líṇaṇauṭu
  1. L は語頭なので l
  2. Í は í
  3. Ŋ は ṇ
  4. 母音字が付加されていないので a
  5. Ɲ は ṇ
  6. Ò は au
  7. T には retracted の記号がついているので ṭ
  8. U は u
ゆえに líṇaṇauṭu
このように、boríṣán báni で書くと、
  • アイル nyautu からも見てとれるように、基底としては //ɲawtu// を持っていると見做せる
  • バート語 nautu は語頭制約により //ɲ// が前進したもの
  • バート語 líṇaṇauṭu は ṇ に引きずられて t が ṭ へと後退したもの
ということが分かりやすい。

逆接の呼応構文 cun ... ja と逆接の文接続詞 ṣiṇṇa

  1. C は c
  2. U は u

  1. J は j
  2. 母音字が付加されていないので a
ゆえに ja

  1. C には retracted の記号がついているので ṣ
  2. U には advanced の記号がついているので [ɨ]
  3. NЪJ には二重線が引いてあるので長い鼻音。J の調音位置によりこれは ṇṇ

ゆえに ṣiṇṇa（反り舌のあとの [ɨ] は i と綴る。）
このように boríṣán báni で書くと、もともとは呼応の副詞であった cun と ja がくっついて ṣiṇṇa「しかし」を形成したことが分かりやすい。
なお、U̟ と綴るべき [ɨ] はこの語ぐらいしかない。cuṇṇa として一語化したのち、頻用であるがゆえに舌を「前 (c) → 後ろ (u) → 中間 (ṇ)」と動かす動作が省略されたことでできた例外的な音形である。
